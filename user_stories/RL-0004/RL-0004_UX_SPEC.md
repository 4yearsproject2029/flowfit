## User Goal

```text
Users need to understand that RepLog works without an account or network connection, and that MVP data is local to the device.
```

---

## Messaging Placement

| Surface | Message Purpose |
| ------- | --------------- |
| Onboarding | Explain before the first saved weekly goal that data is local, no account or network is required, and uninstalling or switching devices may cause data loss. |
| Home Screen | Keep the same local-data limitation visible for returning users. |
