# FlowFit

FlowFit is a Flutter workout tracker MVP built to make workout logging fast, simple, and usable during exercise.

## Implemented Features

- Weekly calendar
- Date-based workout logs
- Add workout bottom sheet
- Workout categories
- Sets, reps, and weight inputs
- Completion checkbox
- Delete workout
- Hive local persistence
- Rest timer

## Tech Stack

- Flutter
- Dart
- Hive
- Material 3

## Planned / Not Currently Used

- Riverpod

## How to Run

```bash
flutter pub get
flutter run
flutter test
```

## Known Limitations

- No authentication
- No cloud sync
- No analytics or charts
- No notifications
- No workout templates yet
